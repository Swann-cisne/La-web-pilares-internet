El Viaje de una Solicitud

Para entender cómo funciona Internet, debemos analizar el recorrido que hace cada consulta desde tu dispositivo hasta el servidor. Estos son los tres pilares de ese viaje:

Cliente - Servidor

Tu navegador (Chrome, Edge) es el Cliente que inicia la petición. La computadora remota es el Servidor que procesa la orden y devuelve los archivos de la web.

Servidor DNS

Las computadoras solo entienden números. El DNS funciona como el directorio telefónico de Internet: traduce nombres como google.com en direcciones IP numéricas.

Dirección IP

Es una etiqueta numérica única asignada a cada dispositivo. Funciona como la dirección residencial de tu casa, permitiendo que los paquetes de datos sepan exactamente a dónde llegar.
